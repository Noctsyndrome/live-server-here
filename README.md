# Live Server Here

Live Server Here is a modern desktop application designed to simplify the process of launching local static servers. By integrating with the Windows context menu, you can instantly start a local web server for any folder with a single click.

![License](https://img.shields.io/badge/license-MIT-blue.svg)
![Version](https://img.shields.io/badge/version-0.1.0-green.svg)

## Features

- **Context Menu Integration**: Right-click any folder in Windows Explorer and select "Live Server Here" to start serving immediately.
- **Modern Dashboard**: Intuitively view ports and status of all currently running servers.
- **History Management**: Automatically records launched folders for quick restart or removal.
- **Favorites**: Pin frequently used projects to your favorites for instant access.
- **Real-time Status**: System tray icon for management and sidebar badges showing the count of active servers.
- **Multi-language Support**: Built-in English and Chinese interfaces.
- **Beautiful UI**: Frameless window design with a tech-inspired grid background and fluid interactions.

## Usage Guide

1. **Start a Server**:
   - **Method 1**: Right-click any folder in Explorer -> "Live Server Here".
   - **Method 2**: Click "Add Folder" in the main dashboard to select a directory.
   - **Method 3**: Click the play button from your History or Favorites list.

2. **Manage Servers**:
   - Click "Stop" on the dashboard to shut down specific servers.
   - Right-click the system tray icon to quickly stop servers or quit the application.
